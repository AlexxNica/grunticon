What is Grunticon?

Grunticon is a Grunt.js task that makes it easy to manage icons and background images for all devices, preferring HD (retina) SVG icons but also provides fallback support for standard definition browsers, and old browsers alike. From a CSS perspective, it's easy to use, as it generates a class referencing each icon; simple as that!

Can I animate them?

Is a uniconkey grumpy?! As of version 2, Grunticon now gives you the option to embed an icon into your markup so you can write JavaScript or CSS directly against the SVG markup–the full power of SVG is now at your disposal!

Check out these icons with a little animation applied!

Neat! So it only works in new browsers then, right?

Wrong! Grunticon generates fallback PNG images for you icons and makes sure those are used in any browser that doesn’t understand SVG. Go ahead, check out this page in IE6 if you’d like. You won’t see enhancements applied to the icons with CSS and JavaScript, but you’ll definitely get a static icon, and that’s often critical to the usability of your site.
